Правила игры

Нужно разработать алгоритм, с помощью которого, робот сможет выбраться из лабиринта (выход -> портал).

Поля F1, F2, M - это пространство в котором можно пиисать программу для робота. Используйте действия расположенные влевой части экрана (вперед, влева, вправо, f1, f2), для составления алгоритма.


F1 - это первая функция, которую можно в дальнейшем использовать в поле M.

F2 - это вторая функция, которую можно в дальнейшем использовать в поле M.

Поле M - это основной алгоритм робота.
