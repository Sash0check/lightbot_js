Правила игры

Чтобы пройти уровень, необходимо провести AR-Bot’а до финишной точки

Для этого заполните основную функцию соответствующими командами

Также вы можете использовать вспомогательные функции для повторяющихся действий, чтобы разгрузить основную функцию.

Порой будет необходимо пройти чекпоинты для того чтобы открыть финиш

Также вы можете собирать монетки для увеличение финального счета.
